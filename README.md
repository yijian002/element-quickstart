# element-quickstart

[github/element-quickstart](https://github.com/yijian002/element-quickstart)

## How use

``` bash
# 如果未安装
$ npm install -g vue-cli

# 开始
$ vue init yijian002/element-quickstart my-project

$ cd my-project
$ npm install
$ npm run dev
```
