# element-quickstart

This is fork from [element-starter](https://github.com/ElementUI/element-starter)

## How use

``` bash
# 如果未安装
$ npm install -g vue-cli

# 开始
$ npm i element-quickstart
$ vue init ./ my-project

$ cd my-project
$ npm install
$ npm run dev
```
